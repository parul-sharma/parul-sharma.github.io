  • Naiive Bayes Classifier for gene expression data

    Mar’17-Apr'17

    Advisor: Dr. Vibhor Kumar

    A Naive Bayesian classifier was developed which categorized gene-expression dataset into Control and Case class. The data was normalized to fit the range [0,1] and gaussian distribution was used for calculating the posterior probability. The accuracy of the classifier for the given dataset was 60 - 70% keeping in account that the test and training data was randomly selected from the corpus. An approximation which did not allow R to approximate very small values to 0 or large values to infinity was chosen which increased the accuracy between 70-80%. Feature selection using PCA was done and the top 10 features were used which showed no significant imporvement in the accuracy as the dataset was very noisy.

  • An online tool for reporting of ACMG approved incidental findings in sequencing data

    Aug’16-Jan’17

    Advisor: Dr. Vinod Scaria

    In 2011, American College of Medical Genetics (ACMG) formed a working group with the main aim of promoting standardized reporting of sequencing data and announced essential reporting of incidental/secondary findings in sequencing data. The working group published a list of genes, and announced the reporting of all variants from the genes irrespective of its relevance for which the sequencing was performed. The tool checks if a sequencing data has variants falling on any of the ACMG provided minimum list of 59 genes from a manually curated database. It then provides useful gene-disease annotations of the matched variants categorised into pathogenic, likely pathogenic and uncertain significance, in a comprehensive report format. The user can select the variants he/she wants to print in the report from the list of matched variants.

  • Protein simulation using structure modelling tool- SMOG server

    Oct’16-Nov’16

    Advisor: Dr. Arnab Bhattacherjee

    The smog-server webtool interface provides all the necessary information for performing structure-based simulations of proteins, nucleic acids, as well as many supported ligands. In addition, the webtool can handle all preprocessing steps that are necessary to prepare a structure-based forcefield for a given molecular structure. The tool was used to simulate the dynamics of PDB-Id=1BC8 protein in absence of water and DNA molecules. The simulations were recorded at different temperatures and the resulting data was used to calculate the folding temperature of the protein.

  • BAX activation in pre-mitochondrial apoptosis pathway using Monte Carlo simulations

    Mar’16-Jul’16

    Advisor: Dr. Subhadip Raychaudhuri
